# sistema-gestion-peliculas-favoritas
Sistema en Python : Pequeña aplicación para agregar, listar y eliminar películas favoritas usando POO
Qué es este proyecto?
Es un programa sencillo diseñado para que un usuario pueda gestionar su propia colección de películas preferidas. Permite llevar un registro organizado, evitando que se repitan títulos y manteniendo un control claro de cada película mediante un código de identificación (ID) único.

¿Qué puedes hacer con él?
El sistema ofrece tres funciones principales para que el usuario controle su lista:

Añadir a favoritos: Guarda una película en tu lista personal. El sistema revisa automáticamente que la película no esté ya guardada y que tenga un número de identificación válido.

Ver mi lista: Muestra de forma ordenada todas las películas que has guardado hasta el momento.

Quitar de favoritos: Permite eliminar una película de la lista si ya no deseas tenerla allí.

Mis_favoritos: Es el "cerebro" o administrador que hace que todo funcione, ejecutando las órdenes de agregar, listar o eliminar.
